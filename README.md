# sandun-abey.github.io

![Portfolio Website](https://sandun-abey.github.io/)

This repository hosts the source code for my personal portfolio website, accessible at [https://sandun-abey.github.io/](https://sandun-abey.github.io/). It is designed to showcase my projects, skills, and experiences.

## Features

- Responsive and user-friendly portfolio design
- Built using HTML (primary language)
- Hosted via GitHub Pages for free, fast, and reliable access

## Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/sandun-abey/sandun-abey.github.io.git
   ```

2. **Explore and customize:**
   - All source files are in the root directory.
   - Edit the HTML files as desired to personalize your portfolio.

3. **View Locally:**
   - Simply open the HTML file in your browser or use a local web server for development.

## Deployment

This site is automatically deployed using GitHub Pages from the `master` branch upon push.
